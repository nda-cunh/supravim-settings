
# Vous faites une autre langue que C, C++, Vala ?

il suffit de faire ``:LspInstallServer <optionel: non dun server>``

la listes des serveur est ici https://github.com/mattn/vim-lsp-settings#supported-languages


je recommande egalement d'ajouter dans votre balise YourConfig:

## Python

```
suprapack add plugin-python-lsp
```

## TypeScript

pour un support typescript plus interessant:
```
suprapack add plugin-typescript-lsp
```

## Vala

```
suprapack add plugin-vala-lsp
```
## Meson Build

note: supramake support meson.build comme constructeur
```
suprapack add plugin-meson-lsp
```
## BluePrint

```
suprapack add plugin-blueprint
```

## ASM (Nasm)

```
suprapack add plugin-asm-lsp plugin-nasm-linter
```
